História do HTML

Início

HTML é uma linguagem de marcação de hipertexto utilizada para criar páginas web. Ela foi desenvolvida por Tim Berners-Lee na década de 1990 e seu objetivo original era facilitar a criação e compartilhamento de documentos científicos online no CERN, onde Berners-Lee trabalhava.

Documentos HTML podem ser interpretados por navegadores (Browser)*, como por exemplo:

  • Google Chrome: Desenvolvido pelo Google
  • Mozilla Firefox: um navegador de código aberto
  • Microsoft Edge: Anteriormente conhecido como Internet Explorer, é o navegador da Microsoft. É baseado no projeto de código aberto Chromium, o mesmo que sustenta o Google Chrome.
  • Safari: Navegador da Apple, projetado para seus dispositivos, como Mac, iPhone e iPad.
  • Ópera: Ele oferece recursos exclusivos, como VPN integrado e bloqueador de anúncios.

O HTML 2.0 foi lançado em 1995 e o HTML 3.2 em 1997 introduziu tabelas e formulários. No entanto, a versão mais significativa foi o HTML 4.01, lançado em 1999, que estabeleceu muitos dos recursos usados ainda hoje.

Após o HTML 4, uma grande mudança ocorreu com a introdução do XHTML*, que seguiu as regras do XML*. No entanto, o XHTML não conseguiu substituir completamente o HTML na prática.

A versão mais recente é o HTML5, lançado em 2014, introduziu inúmeras melhorias e novos recursos, como suporte para reprodução de áudio e vídeo, gráficos vetoriais, melhor semântica e APIs para interatividade avançada.

O que são "Tags"

Início

HTML usa “marcadores” ou “tags” para definir elementos em uma página da web, como títulos, parágrafos, links, imagens e muito mais. Elas são cercadas por colchetes angulares (< >), com uma tag de abertura e uma tag de fechamento. Segue abaixo alguns exemplos:

  • "strong" – Negrito
  • i - Itálico
  • u - Sublinhado
  • mark - Marca Texto
  • sup - Adiciona informação em cima da palavra
  • hr - Adiciona uma linha de separação
  • small - Cria um texto pequeno
  • font - Definir estilo de fonte, Cor, Tamanho
  • del - Mostra o texto que foi deletado
  • abrr - Mostra informação da palavra abreviada, quando passa mouse em cima
  • input - Campo de formulário
  • img - Adicionar Imagem
  • a - Linkar uma página com a outra (Hipertexto)
  • h1, h2, h3, h4, h5, h6 - Título e Subtítulo
  • p - Paragrafo (Texto)
  • blockquote - Citação de texto ou observação
  • ol - Lista Ordenada (Sequência 1. 2. 3.)
  • ul - Lista não ordenada (●)

O que são Atributos

Início

São informações adicionais que você pode adicionar às tags para modificar ou fornecer mais detalhes sobre como um elemento deve se comportar. Exemplo:

  • id: Identifica um elemento na página.
  • class: Associa um elemento a uma ou mais classes CSS, permitindo a aplicação de estilos específicos.
  • style: Permite a inclusão de estilos CSS diretamente no elemento.
  • src: Especifica a fonte (source) de elementos como imagens, scripts ou vídeos.
  • alt: Fornece um texto alternativo para elementos como imagens, útil para acessibilidade.
  • href: Utilizado em elementos de âncora () para definir o destino do link.
  • width e height: Usados para definir a largura e a altura de elementos
  • disabled: Desativa um elemento, como um botão ou um campo de formulário.
  • Target: Definir onde o link vai ser aberto (_self = na mesma página ou _blank = outra pagina)

Tipos de Ferramentas

Início

Você pode escrever HTML em qualquer lugar, desde que tenha acesso a um editor de texto, para criar e salvar arquivos com extensão ".html". Porém existem várias ferramentas para auxiliar, desde editores de código até recursos de inspeção e validação.

Segue alguns exemplos:

  • Editores de Código: Visual Studio Code, Eclipse
  • Plataformas Online:JSFiddle, CodePen e Replit
  • Editores de Texto:Bloco de Notas, Notepad++, TextEdit, Gedit
